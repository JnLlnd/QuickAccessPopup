Quick Access Popup
==================

Absolutely free to download and use, for personal or commercial use.


Easy one-step installation
--------------------------

1) Download, execute the quickaccesspopup.exe file and just follow the setup instructions.

2) Enjoy!

Note: If an anti-virus or protection software reports Quick Access Popup as a potential threat,
this is not the case. Please read this post: Should I use QAP if an anti-virus is blocking its
download or its execution? (www.quickaccesspopup.com/anti-virus-blocking-download-or-execution/).


Portable installation
---------------------

Quick Access Popup is also available for "portable" installation. This will, for example, allow
you to run it from an external drive or USB key without have to install it on the host computer.
Download the zip file quickaccesspopup.zip and follow these instructions.
